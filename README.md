# RingBuffer
## Testing Environment Information
OS: Ubuntu 22.04\
g++: 11.4.0\
GNU make: 4.3

## Install Required Build Tools
```
sudo apt install g++ make
```

## Clone Repository

```
git clone https://github.com/JThomas598/RingBuffer.git RingBuffer && cd RingBuffer
```

## Compile Code
```
make
```

## Run Program
```
./app <num iterations> <buffer size>
